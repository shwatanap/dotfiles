# asdf
## プラグインの管理
### 追加済みのプラグイン一覧確認
asdf plugin-list

### プラグインの追加
asdf plugin-add \<name\>

### プラグインの削除
asdf plugin-remove \<name\>

## 言語の管理
### 一覧確認
asdf list

### 特定の言語の確認
asdf list \<name\>

### 言語バージョン追加
asdf install \<name\> \<version\>

### 言語のバージョン削除
asdf install \<name\> \<version\>

### グローバルバージョン指定
asdf global \<name\> \<version\>

### ローカルバージョン指定
asdf local \<name\> \<version\>

## .tool-version
### 一括インストール
asdf install
